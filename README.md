# Discreta II

Material y códigos de la materia Matemática Discreta II.

## Estructura del repositorio

| Carpeta | Contenido |
|---------|-----------|
| **docs/referencia** | Apuntes, artículos y material de referencia (PDF) |
| **docs/enunciados** | Enunciados de prácticos (PDF) |
| **codigos/** | Códigos por práctico: `disc2` (prácticos 1–8), `codigos_discreta`, `discreta_ii` |
| **algoritmos/** | Implementaciones de flujo máximo: `edmonds-karp`, `edmonds-karp2`, `dinic` |
| **lab/** | Trabajo práctico de laboratorio (informe, ejemplos) |
| **final/** | Material y código del final |
| **ayudantia/** | Material de ayudantía |
| **logs/** | Salidas de consola y mediciones de tiempo |

## Notas

- `codigos/disc2` y `lab` tienen su propio repositorio git (.git). Si trabajás desde ahí, podés tratarlos como submódulos o copias independientes.
- Las carpetas que venían de Subversion (`.svn`) están ignoradas en `.gitignore`.
